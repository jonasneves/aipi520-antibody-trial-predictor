Project Overview

Real-world ML challenge: predicting clinical trial success for antibody therapeutics

Problem Statement i Why Antibody Trials?
Antibodies represent ~30% of the pharmaceutical development pipeline with high commercial value ($237B+ market), but face historically high failure rates.

Predict probability of success for Phase 2 & 3 antibody clinical trials. With 18.2% failure rate observed in our dataset and costs of $10-100M+ per trial, early prediction can save pharmaceutical companies significant resources.

Data Source

ClinicalTrials.gov Bulk XML Download

Streaming parse from S3-hosted bulk XML archive

Phase 2/3 interventional antibody trials (completed/terminated/withdrawn status)

Success Labeling Strategy

Refined Binary Classification:

  • Success (1): COMPLETED, APPROVED_FOR_MARKETING, AVAILABLE
  • Failure (0): TERMINATED/WITHDRAWN/SUSPENDED for efficacy or safety reasons
  • Excluded: Trials terminated for administrative reasons (funding, enrollment, business decisions)

This approach focuses on efficacy outcomes rather than administrative completion.

Feature Engineering - 32 Features

Domain-specific feature set:

Antibody Type & Mechanism (12 features) Trial Design & Enrollment (8 features) Disease Categories (8 features) Sponsor Characteristics (3 features) Temporal Features (4 features + has_start_date flag) Trial Metadata (1 feature)

Antibody-specific features: Type classification (murine/chimeric/humanized/fully_human), target mechanisms (checkpoint inhibitors, growth factors, cytokines, CD markers), biomarker selection, combination therapy indicators

Methodology: Post-2024 trials filtered (22), text features excluded, has_start_date flag for missing dates (45%), stratified random split
