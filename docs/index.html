Project Overview

Real-world ML challenge: predicting clinical trial success for antibody therapeutics

Problem Statement i Why Antibody Trials?
Antibodies represent ~30% of the pharmaceutical development pipeline with high commercial value ($237B+ market), but face historically high failure rates.

Predict probability of success for Phase 2 & 3 antibody clinical trials. With 70% failure rate in Phase 2 and costs of $10-100M+, early prediction can save pharmaceutical companies significant resources.

Data Source

ClinicalTrials.gov API v2

Completed Phase 2/3 antibody trials

AREA[OverallStatus]COMPLETED AND (AREA[Phase]PHASE2 OR AREA[Phase]PHASE3)

Success Labeling Strategy

Binary Classification Based on Trial Status:

  • Success (1): COMPLETED, APPROVED_FOR_MARKETING, AVAILABLE
  • Failure (0): TERMINATED, WITHDRAWN, SUSPENDED, NO_LONGER_AVAILABLE
  • Excluded: Ambiguous statuses

Dataset Statistics

Sample Counts

  • Total Samples: 50,000 trials
  • Training Set: 40,000 samples (80%)
  • Test Set: 10,000 samples (20%)
